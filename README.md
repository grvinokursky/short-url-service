# Итоговый проект "Сервис коротких ссылок"

## Описание

Сервис сокращения ссылок с возможностью управления лимитом переходов и времени жизни ссылки.

Взаимодействие с пользователем осуществляется через интерфейс командной строки.

## Быстрый старт

1. Убедитесь что на вашем компьютере установлен JDK 25.
2. [**Скачайте по ссылке**](https://drive.google.com/file/d/1ZBuZUshPLX4A7k5NjZlICNxJFL-IdSiQ/view?usp=sharing) и распакуйте на своем компьютере архив с приложением.
3. Из директории архива выполните команду `java --enable-native-access=ALL-UNNAMED -jar short-url-service-1.0-SNAPSHOT.jar`

Ожидается что кодировка вашей консоли будет UTF-8. Иначе в вашей консоле вместо русских символов будут 'кракозябры'. Чтобы перевести открытую страницу консоли в необходимую кодировку введите команду: `chcp 65001` (для Windows).

Альтернативный вариант запуска приложения через IntelliJ IDEA.

## Управление

### Авторизация

Выполните авторизацию для входа в сервис генерации коротких URL.

Доступные команды:

- `register (reg) <username>` - Зарегистрировать пользователя.
- `login (l) <username>` - Войти как пользователь.

### Работа с сервисом коротких ссылок

Доступные команды:

- `list` - Показать ссылки текущего пользователя.
- `create <baseUrl>` - Создать короткую ссылку (опционально -maxClicks <maxClicksValue>).
- `open <shortUrl>` - Открыть короткую ссылку.
- `exit` - Выйти из приложения.

## Описание решения

### Архитектура решения

При проектировании решения было предусмотрено разделение на следующие слои:

- слой данных,
- слой сервисов,
- слой приложения.

#### Слой данных

Слой данных включает модели данных, контейнер для хранения данных и репозитории для инкапсуляции взаимодействия с хранилищем данных.

#### Слой сервисов

Данный слой включает в себя сервисы, заключающие бизнес-логику приложения. Также для ответов сервиса управления финансами были созданы модели отчетов.

#### Слой приложения

Слой приложения включает пользовательское CLI-приложение.

### Система сборки

В качестве системы сборки используется Gradle.

Решение использует Spring Framework.

### Генерация коротких ссылок

Для генерации коротких ссылок используется сервис Яндекс Кликер.
